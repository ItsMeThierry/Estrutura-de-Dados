# Fila (Array)

Atividade para a disciplina Estrutura de Dados - UFPB.

Implementação de uma fila encadeada de inteiros com as operações:

- Verificação de fila vazia ou cheia;
- Obter tamanho da fila;
- Obter o valor do inicio da fila;
- Inserir um elemento na fila;
- Remover um elemento na fila;
- Imprimir os elementos da fila.

O programa em si funciona como um CRUD, onde o usuário poderá escolher as operações baseadas com as opções do menu. Entretanto, todo o código relacionado à estrutura estará no arquivo 'queue.c'.
